Visualization

TGX provides access to several plots as visualizations of the temporal dynamics in a temporal graph.

Function Parameters ?
nodes_and_edges_per_ts graph: list, plot_path: str = None, network_name: str = None
nodes_per_ts graph: list, plot_path: str = None, network_name: str = None
edges_per_ts graph: list, plot_path: str = None, network_name: str = None
num_connected_components_per_ts graph: list, plot_path: str = None, network_name: str = None
average_degree_per_ts graph: list, plot_path: str = None, network_name: str = None
TEA graph: list, plot_path: str = None, network_name: str = None
TET graph: list, plot_path: str = None, network_name: str = None
degree_density graph_edgelist, k = 10, network_name: str = None, plot_path: str = None

For each visualization tool, you can specify a path as input that will be used to save the plot.

Example: tgx.nodes_and_edges_per_ts(data, network_name=dataset.name, plot_path = your_path)

Number of nodes and edges plot

Plot the number of nodes per timestamp and number of edges per timestamp in one fiugre.

In[]:
Copied! 
tgx.nodes_and_edges_per_ts(data, network_name=dataset.name)
tgx.nodes_and_edges_per_ts(data, network_name=dataset.name)
Tip: Use ```nodes_per_ts``` and ```edges_per_ts``` to individually plot the number of nodes or edges, respectively.

Statistics

TGX provides several tools that are helpful in the statistical analysis of temporal graphs.

| Function | Description | Parameters | Returns | --- | --- | --- | --- | | get_reoccurrence | Calculate the recurrence index | | List[float] | | get_surprise | Calculate the surprise index | | List[float] | | get_novelty | Calculate the novelty index | | List[float] | | get_avg_node_activity | Calculate the average node activity | | List[float] | | get_avg_node_engagement | Calculate the average node engagement | | List[float] | | size_connected_components | Calculate the sizes of connected components | | List[List[float]]|

You can use plot_for_snapshots for visualizing the statistics functions.

Important: A discretized graph should be provided as input.

Average Node Engagement

Calculate the average node engagement over time. Node engagement represents the average number of distinct nodes that establish at least one new connection during each time step.

In[]:
Copied! 
node_engagement = get_avg_node_engagement(data)
plot_for_snapshots(node_engagement, filename=f"{dataset.name}_avg_node_engagement_per_ts", y_title="node engagement")
node_engagement = get_avg_node_engagement(data) plot_for_snapshots(node_engagement, filename=f"{dataset.name}_avg_node_engagement_per_ts", y_title="node engagement")
